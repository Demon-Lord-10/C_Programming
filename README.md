# C Programming

Working my way through **The C Programming Language** by Brian W. Kernighan & Dennis M. Ritchie — solving exercises chapter by chapter.

---

## Compile & Run

```bash
gcc filename.c -o output
./output
```

For programs that take input, press **Ctrl+D** (Linux/macOS) or **Ctrl+Z** (Windows) to signal end of input.
